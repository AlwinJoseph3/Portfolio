## What makes this different

This wasn't built with a template. Every component, animation, and 3D scene was written from scratch with a focus on performance — not just visuals.

The result: a **Lighthouse Performance Score of 96** while running live Three.js 3D scenes and physics-based transitions.

---

## Features

- 🌐 **Interactive 3D components** — Three.js scenes integrated directly into the UI
- 🎯 **Physics-based UI transitions** — motion that responds and feels alive
- ⚡ **Lighthouse Performance: 96** — optimised rendering, lazy loading, minimal bundle
- 🔍 **SEO optimised** — semantic HTML5 structure and custom meta tags
- 📱 **Fully responsive** — consistent experience across all screen sizes
